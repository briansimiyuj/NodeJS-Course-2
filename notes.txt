A `middleware` is a code that runs on the server between getting the request and sending the response.
We can have multiple middlewares in a single route.
They run from top to bottom.

Examples Of Middleware:
   1. Logger middleware to log every request.
   2. Authentication middleware to check if the user is authenticated or not.
   3. Middleware to parse JSON data from the request body.
   4. Middleware to return a 404 error if the requested resource is not found.
