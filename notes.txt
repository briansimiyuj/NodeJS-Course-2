Node JS has a global object called `global`. It works like `window` in the browser. 

`global` object has a lot of properties and methods. For example, `setTimeout`, `setInterval`, `clearTimeout`, `clearInterval`, `console`, `process`, `Buffer`, `__dirname`, `__filename`, etc.

`__dirname` and `__filename` are special properties of the `global` object. They give us the absolute path of the current directory and the current file respectively.
