When we type an URL into the browser, the browser sends a request to the server for the page. The server responds data to the browser. Data can be HTML, CSS, JS, Images, Resources, etc.

Each computer has an unique address that identifies it on the internet. This address is called IP address. 

IP address is just a sequence of numbers separated by periods. For example, `192.168.1.1` is an IP address. It's very difficult to remember IP addresses. So, we use domain names instead of IP addresses. For example, `google.com` is a domain name. When we type a domain name into the browser, the browser converts it into an IP address.

Once the browser gets the IP address, it sends a request to the server for the page. The server responds with the page data. The browser then renders the page on the screen. This happens via HyperText Transfer Protocol (HTTP).
