Sometimes we might split our code into multiple files, and we want to import functions from one file to another. This is called a module.

If we want to use a function or variable from another file, we need to export it first. We can do this by using the `export` keyword.
    `export const ${functionName} = () => { ... }` or `export const ${variableName} = 42;`

Then, in the file we want to import the function or variable from, we can use the `import` keyword.
    `import { ${functionName}, ${variableName} } from `${filePath}.${extension}`

If we want to export multiple functions or variables, we can use write the variables and functions in normal way, and then use `export { ${functionName}, ${variableName} }` at the end of the file.
