There are main types of Databases: SQL and NoSQL

SQL is a structured query language that is used to manage data in a relational database management system (RDBMS). It uses tables to store data and allows for complex queries and relationships between data.

NoSQL is a non-relational database management system that is used to store and manage large amounts of unstructured data. It uses collections of documents. Each collection contains one type of document, and each document can have a different structure. The document contains key-value pairs.

An example of a NoSQL database is MongoDB, which is a document-oriented database that stores data in JSON-like documents.
