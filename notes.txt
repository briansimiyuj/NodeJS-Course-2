A mongoose is an Object Document Mapping (ODM) library for MongoDB. It wraps MongoDB driver functionality and makes it much easier to work with. It allows us to create simple data models which have queries to perform CRUD operations on the database.

Schema defines the structure of the document and the types of the data. It is used to create a new model. For example, we can create a schema for a user with an ID, name, email, and password. For a blog, we can create a schema with an ID, title, snippet and content.

A model allows us to perform CRUD operations on the database. For example, we can create a new user, update an existing user, delete a user, and find a user by their ID.

To install mongoose, we can use:
    `npm install mongoose`
