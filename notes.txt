To delete a file, we need to check if it exists and then delete it. We use `fs.existsSync(${filePath})` to check if the file exists. If it does, we use `fs.unlink(${filePath})` to delete it. `unlink` method takes two parameters: the file path and a callback function

`if(fs.existsSync(${filePath})){

    fs.unlink(${filePath}, (err) => {
        if (err) throw err;
        console.log(`${filePath} was deleted`);
    })

}`
