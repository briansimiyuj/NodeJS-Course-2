Express router helps us to split our application into different modules. Let's create a file called "blogRoutes.js" 
Then we transfer all the blog related routes to this file.
Then we create an instance of express router and assign it to a variable called `blogRouter`.
Then replace all the routes with `app` with `blogRouter`.
   - make sure "/create" route comes before route parameter routes
At the bottom of the file we export the blogRouter.

In `express.js` we import the blogRouter and use it as a middleware.
   `app.use(blogRouter)`

We can add another argument to `app.use()` to specify which path to use the middleware on.
   `app.use("/blogs", blogRouter)`

Then we can change the routes in `blogRoutes.js` from "/blogs/..." to "/...".
