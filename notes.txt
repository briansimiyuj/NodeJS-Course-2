Node JS can read, write, create, and delete files using the `fs` module.

We can use the `readFile` method on the `fs` module to read a file. This method takes two arguments: the relative path to the file and a callback function that will be called when the file is read. The callback function takes two arguments: an error object and the data read from the file. Before the callback function, we need to specify the encoding of the file. If we don't specify the encoding, the data will be returned as a buffer.

`fs.readFile("${filePath}", ${fileEncoding}, (err, data) =>{

    do something with the data      
    
})`

This is asynchronous code, meaning that the code will continue to run while the file is being read. Refer to `script.js`; 'Last line' is printed before the file is read.
