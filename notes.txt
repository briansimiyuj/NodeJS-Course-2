We can create a file that can be shared between the views. This is called Partial Views. This can be the header, navbar, footer, etc.

Let's create a partial view for the navbar. We do the markup for the navbar only without the html tags. We can then use this partial view in our views by using `include(${filePath})` ${filePath} is the relative path to the partial view.
